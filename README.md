# Cart Assignment

This is a Vite project using Typescript,React 18.2 and TailwindCss.

## Getting Started

First, install the module libraries:

```bash
npm  install
```

Second, run dev mode:

```bash
npm  run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
